# LUU Open

Codex skills and small agent workflows by LUU.

LUU 维护的 Codex skills 与轻量智能体工作流。

This repository collects reusable packages for Codex. Each top-level folder is one package with its own README.

本仓库收集可复用的 Codex 工具包。每个一级目录都是一个独立包，并包含自己的 README。

## Packages / 包

- [Compact Better](compact-better/README.md): review and prepare a temporary compact prompt before manually running Codex `/compact`.
  - 中文：在手动执行 Codex `/compact` 前，审阅并准备临时压缩提示。

## Layout / 结构

```text
compact-better/
  README.md
  .codex-plugin/plugin.json
  skills/
```
